# Quarry Design Project – Sandstone Crusher (500m × 500m)

Student: Osama Almsaafeh  
Major: Mining Engineering  
University: Tafila Technical University  

## Project Description
This project presents the design of a sandstone quarry including:

- Quarry layout design
- Bench geometry
- Haul road design
- Drilling pattern
- Blasting parameters
- Crusher location
- Reserve estimation

## Software Used

AutoCAD
Excel
Google Earth
QGIS (optional)

## Project Area

500m × 500m sandstone quarry
